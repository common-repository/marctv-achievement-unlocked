== Description ==

Displays a graphical achievement message at the bottom of an article with a custom field "achievement" set. Works on iPhone, iPad, Chrome, Safari, Firefox but looks ugly on IE7+.

== Installation ==

Upload the "MarcTV Achievment unlocked" plugin to your blog, Activate it.

When editing an article just create a custom field called "achievement" with your desired text to be displayed as text for the achievement message. If the user reaches the bottom (#footer) he will be greeted with a special message.
